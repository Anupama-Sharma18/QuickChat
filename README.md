# 💬 QuickChat – Chat Application

QuickChat is a lightweight browser-based chat application built using **HTML, CSS, and JavaScript**.
It allows users to register, log in, join chat rooms, and send messages in a simple and interactive interface.

This project demonstrates **frontend development skills, DOM manipulation, and browser storage usage**.

---

## 🚀 Features

* 🔐 User Registration and Login
* 💬 Multiple Chat Rooms (General, Tech, Random, AWS Cloud)
* 💾 Persistent Messages using **Browser LocalStorage**
* 👤 Profile Editing (Change Name & Profile Photo)
* 🟢 Online / Offline User Indicators
* 😀 Emoji Support in Messages
* 👥 Members List with Status
* ✏️ Profile Avatar Upload & Preview
* 📱 Responsive UI Design

## ⚙️ How It Works

1. User creates an account using the **Register page**.
2. Login credentials are stored in **localStorage**.
3. After login, the user enters the **chat interface**.
4. Messages are saved in **localStorage per chat room**.
5. Members list displays **online/offline users**.
6. Profile settings allow users to **change name and avatar**.
